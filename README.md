# GCE VECTREX

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **EXEREÚNESIS**.

GCE VECTREX (VEC) — a UD0 EXEREÚNESIS teardown + spec edition. Unique among consoles, the Vectrex came with its own built-in vector monitor — it drew crisp glowing lines directly, not a grid of pixels, so it never needed a TV. A Motorola 6809 steered the beam through a DAC, and plastic overlays added colour to the black-and-white screen.
